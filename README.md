Are you bored of waiting ur addons getting reinstalled because you tested your map made in Hammer?

Are you bored of setting ur addoninfo as read-only?

Its WASTING YOUR TIME right?

Well. No worries.
This Addon list changer, could fix the time problem so fast.
Heres how to use it:
1. Copy the common/Left 4 Dead 2/left4dead2 directory (not common/Left 4 Dead 2)
2. Open the AddonListDirectory.cfg
3. And paste it inside
4. And then. Run the Addonlist Changer.bat file

now you have choosen options there. Which it explains it all:

Type 1: Turn off all addons (Feature)
Type 2: stop addon getting removed from hammer map testing
Type 3: Back to Normal

If you want to download it. Click Code and Download as ZIP

After that. Have fun editing yo map!
